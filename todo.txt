File Extensions
---------------

All files will end in the extension ".nars".  There will be a secondary
extension to the left of the global extension such as .ws.nars for workspaces,
(e.g. porsche.ws.nars).

Workspaces              .ws.nars
File Systems            .file.nars
Plugins                 .plugin.nars
Initialization File     .ini
